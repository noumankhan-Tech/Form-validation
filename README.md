# Professional Registration Form

A modern and responsive registration form built with HTML, CSS, and JavaScript. Features real-time validation, password visibility toggle, Font Awesome icons, and a clean user-friendly interface.

## Features

* Username, Email & Password Validation
* Show/Hide Password
* Responsive Design
* Modern UI with Animations
* Font Awesome Icons

## Technologies

* HTML5
* CSS3
* JavaScript
